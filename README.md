# Vivero Online

E-commerce de plantas con **Angular 17** + **NestJS 10** + **PostgreSQL**.

## 🚀 Iniciar en 2 Comandos

```bash
cd /c/Users/maru3/Desktop/PROJECTOS/PRUEBA-KIRO
bash setup.sh
```

Luego abre: **http://localhost**

## 📖 Documentación

- **AGENT.md** - Documentación completa del proyecto
- **setup.sh** - Script de instalación automática

## 🎯 Comandos Rápidos

```
docker-compose up -d              # Iniciar
docker-compose logs -f backend    # Ver logs
docker-compose down               # Parar
docker-compose exec backend bash  # Terminal backend
bash check-setup.sh               # Verificar setup
```

Ver más en **AGENT.md**.
